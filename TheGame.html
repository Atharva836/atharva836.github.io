Unscramble the Word

1. 'Unscramble the Word', In this game you will be given some letters.
2. You have to rearrange those letters and form a meaningful word, using all the letters given.

e.g. The letters are A A V J
The Answer for given question is 'JAVA' !

3. All the other important instructions are presented in the Game.


Guidelines for Playing the Game:

1. Click on the link / button below (TheGame.java)
2. Copy the Entire Text and paste in any Text-Editor
3. Make sure that you have installed JDK
4. Run the JAVA Program and Play!!!
*NOTE* Save the name of the file in your text editor as "Game" only!!
